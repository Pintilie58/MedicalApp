# Images folder

Pune aici imaginea de welcome cu numele **welcome.jpg**

Calea completă așteptată:
`wwwroot/images/welcome.jpg`

Dacă nu pui nicio imagine aici, pagina va afișa automat un placeholder generat online (via.placeholder.com).

Poți folosi orice extensie, dar trebuie să actualizezi și link-ul din `Views/Home/Index.cshtml` dacă folosești altă extensie (ex: welcome.png, welcome.webp).
